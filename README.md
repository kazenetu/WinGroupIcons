# WinGroupIcons
アイコンをグループ化したり解除したりするサンプル

# 開発環境
VisualStudio 2015

# 使い方
### 起動直後の画面  
![起動直後](ss1.PNG)  
- アイコンの追加ボタンをクリックするたびにアイコンが表示されます。  

### アイコン表示  
![アイコン表示](ss2.PNG)  
- アイコンをドラッグしてほかのアイコンに重ねるとグループになります。
- コントロールキーを押しながらクリックすると複数選択が行えます。  

### コンテキストメニュー
![コンテキストメニュー](ss3.PNG)  
- キャンセル  
選択状態をクリアします。
- 削除  
選択したアイコンを削除します。
- グループ設定  
選択したアイコンをグループにします。
- グループ解除  
選択したアイコンのグループを解除します。
